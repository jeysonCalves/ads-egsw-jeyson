# Sistema de Login

Este projeto tem como objetivo desenvolver um **sistema simples de login em Python** utilizando boas práticas de controle de versão com **Git e GitHub**.

##  Descrição do Projeto
O sistema permite que o usuário faça login com nome de usuário e senha. Ele pode ser usado como base para aplicações maiores.

##  Funcionalidades
- Tela de login
- Validação de usuário e senha
- Mensagens de erro para login incorreto
- Simulação de autenticação

##  Tecnologias Utilizadas
- Linguagem: **Python**
- Git e GitHub para controle de versão
